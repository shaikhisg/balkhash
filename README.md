# Мониторинг и прогноз динамики озера Балхаш (2018–2030)

Репозиторий содержит аналитический модуль для оценки состояния водных ресурсов и прогнозирования уровня воды на основе данных дистанционного зондирования земли (Sentinel-2) и спутниковой альтиметрии.

### Основные результаты:
* **Спутниковый анализ**: Использование индекса **NDWI** для выделения акватории и анализа береговой линии.
* **Математическая модель**: Линейный тренд падения уровня составляет ~0.15 м/год.
* **Прогноз**: Ожидаемый уровень к 2030 году — **340.43 м**.
